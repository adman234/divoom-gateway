📟 Divoom Gateway

Turns an ESP32 into a Bluetooth Classic gateway for Divoom pixel-art devices (Pixoo, Ditoo, Timebox, Tivoo…), made for the Divoom Home Assistant integration.

Install firmware

  1. Connect your ESP32 to this computer with a USB data cable.
  2. Click Install and pick the serial port.
  3. After flashing, the installer asks for your WiFi credentials — no code or config files needed.
  4. Open the device's web page (shown after setup) to configure MQTT and other options.

Your browser does not support Web Serial. Use Chrome or Edge on desktop. You must view this page over HTTPS to install.

⚠️ Only the original ESP32 chip works. The ESP32-S2, S3 and C3 do not have Bluetooth Classic, which Divoom devices require.

After installation

The gateway shows up in Home Assistant automatically (zeroconf) once it has discovered your Divoom device via Bluetooth. You can also reach its web interface at http://divoom-gateway.local to check status, change WiFi/MQTT settings, or update the firmware later.

If the gateway can't reach your WiFi, it opens an access point called Divoom-Gateway (password divoom1234) with a captive setup portal.
